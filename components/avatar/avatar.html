Avatar
Avatar can be used to show user's profile picture or their status on different platforms. Simple Avatars can be created using avatar class.Avatars are user profile pictures, usually found in circular shapes.Avatars help people create their own identity.

Size Avatars:

Size of the avatars can be changed by adding size-larger/size-larger class to the existing avatar class.

Square Avatars:

Shape of the avatars can be changed by adding squareclass to the existing avatar class.

border Avatars:

border can be added in avatars using border-green or border-red class to the existing avatar class. you can change the color of the border
