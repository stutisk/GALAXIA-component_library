Alert
The alert takes the focus away from the current window, and forces the user to read the important message.
It prevents the user from accessing other parts of the page until the alert box is closed. To use this alert just use class name alert and also add class according to alert type.

These are the static examples shown below. It can be categorised into two parts , primary and secondary alerts.Simple Alerts can be created using alert class.
